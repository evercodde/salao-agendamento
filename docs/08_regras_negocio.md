# Regras de Negócio

RN01 Somente a empreendedora pode confirmar ou cancelar agendamentos.

RN02 O cliente deve escolher um serviço antes de selecionar o horário.

RN03 Dois agendamentos não podem ocupar o mesmo horário.

RN04 Cada serviço possui duração estimada.

RN05 O sistema bloqueia automaticamente o tempo do serviço na agenda.

RN06 Serviços podem possuir preço variável (a partir de).

RN07 O cliente pode adicionar descrição ao agendamento.

RN08 Lembretes de agendamento devem ser enviados automaticamente.

RN09 Fotos do portfólio só podem ser gerenciadas pela empreendedora.

RN10 Serviços devem estar organizados por categoria.

RN11 O sistema deve considerar intervalo entre atendimentos.
