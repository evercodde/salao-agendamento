# Ideia do Produto  
  
## Nome do sistema  
A definir com a empreendedora.  
  
## Descrição da ideia  
  
Criar um sistema web mobile-first para auxiliar uma empreendedora de salão especializada em tranças, tratamentos capilares e mega hair a organizar sua agenda, apresentar seu portfólio de serviços e permitir que clientes realizem agendamentos online.  
  
O sistema deve reduzir o trabalho manual atualmente feito via WhatsApp e agenda manual.  
  
## Usuários do sistema  
  
Empreendedora (admin)  
  
- gerenciar agenda  
- cadastrar serviços  
- publicar fotos de trabalhos  
- acompanhar atendimentos  
  
Cliente  
  
- visualizar portfólio  
- escolher serviço  
- agendar horário  
  
## Problema principal  
  
A empreendedora atualmente utiliza:  
  
- WhatsApp  
- agenda manual  
- sistema pago limitado  
  
Isso gera:  
  
- conflitos de horário  
- muito trabalho manual  
- dificuldade de mostrar serviços  
- no-show de clientes
