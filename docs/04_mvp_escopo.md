# Escopo do MVP

## O que é MVP

MVP é a versão mínima do produto que já entrega valor para o usuário.

## Funcionalidades do MVP

1 Portfólio público com fotos
2 Cadastro de serviços com preço e duração
3 Agendamento online
4 Cálculo automático de horários disponíveis
5 Painel administrativo com agenda

## Funcionalidades fora do MVP

- IA no WhatsApp
- assistente capilar
- passaporte capilar
- pagamentos online
- múltiplos profissionais
