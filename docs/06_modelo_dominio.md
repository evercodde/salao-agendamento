# Modelo de Domínio

Principais entidades do sistema.

Usuario
Representa administradores do sistema.

Cliente
Representa clientes do salão.

CategoriaServico
Agrupa serviços semelhantes.

Servico
Define os serviços disponíveis.

Agendamento
Representa um horário reservado.

Produto
Produtos utilizados nos procedimentos.

ConsumoProduto
Registro de produtos usados em um atendimento.

FotoPortfolio
Imagens de trabalhos realizados.
