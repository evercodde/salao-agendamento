# 🗺 Roadmap do Produto

Este documento apresenta o **planejamento evolutivo do sistema de agendamento para salão**.

O objetivo é desenvolver o produto em fases progressivas.

---

# 🚀 Fase 1 — Portfólio Digital

Objetivo:

Permitir que clientes visualizem os trabalhos do salão.

Funcionalidades:

* página inicial
* portfólio de fotos
* organização por categoria
* painel admin para upload de fotos

Resultado esperado:

Clientes conseguem conhecer os serviços antes de agendar.

---

# 📅 Fase 2 — Sistema de Agendamento

Objetivo:

Permitir que clientes agendem horários online.

Funcionalidades:

* cadastro de serviços
* duração do serviço
* seleção de data
* cálculo de horários disponíveis
* confirmação de agendamento
* painel administrativo com agenda

Resultado esperado:

Redução do trabalho manual de agendamento.

---

# 🔔 Fase 3 — Lembretes Automáticos

Objetivo:

Reduzir faltas em agendamentos.

Funcionalidades:

* integração com WhatsApp
* envio automático de lembretes
* configuração de antecedência do lembrete

Tecnologias possíveis:

* automação com n8n
* integração com WhatsApp API

---

# 🤖 Fase 4 — Assistente Capilar com IA

Objetivo:

Criar uma assistente digital que conhece o histórico capilar da cliente.

Funcionalidades:

* consulta de histórico de procedimentos
* recomendações de serviços
* respostas automáticas no WhatsApp

Exemplos de perguntas:

```
Quando foi meu último alisamento?
Posso fazer química novamente?
Qual tratamento você recomenda?
```

---

# 🧬 Fase 5 — Passaporte Capilar

Objetivo:

Criar uma plataforma independente de salão.

A cliente terá um **histórico capilar portátil**.

Funcionalidades:

* histórico de procedimentos
* registro de produtos utilizados
* compartilhamento com outros salões
* perfil capilar da cliente

Modelo de negócio possível:

* SaaS para salões
* plano premium para clientes

---

# 📈 Visão de Longo Prazo

Transformar o sistema em uma **plataforma digital para gestão de serviços capilares especializados**.

Possibilidades futuras:

* marketplace de profissionais
* recomendação de produtos
* análise capilar com IA
* gestão de múltiplos salões

Validar se o sistema resolve os problemas reais da empreendedora:

* organização da agenda
* apresentação dos serviços
* facilidade para clientes agendarem
